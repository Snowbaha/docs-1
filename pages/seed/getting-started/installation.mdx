# Installation

## For generating seed data

First, you'll need [Node.js](https://nodejs.org/en/) and [postgres](https://www.postgresql.org/) installed on your system.

Once you have these, run the following in the command line and follow the prompts:

```bash >_&nbsp;terminal
# Set up Snaplet with npx
npx @snaplet/seed init
```
